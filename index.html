For agents (and their humans)

Three front doors, same data: a plain URL for anything that speaks HTTP, an llms.txt for anything that reads, and a typed MCP server for anything that calls tools.

Plain HTTP

Every payload is a static file. Same envelope everywhere: dataset, unit, effective date, sources, data.

curl -s https://cdn.jsdelivr.net/gh/forge-arcana/kod-data@main/fx/v1/usd.min.json \
  | jq .rates.EUR

Public · free

Hot tier — now + 90 days

Current values plus a rolling 90-day window of dated snapshots per dataset (<dataset>/v1/<date>.min.json), served free on this CDN. Git history of any path is the window's revision history.

Private · offered separately

Deep history — the full archive

The complete USD-anchored observation time series — every instrument, field and day, one tidy row each, with annual NDJSON.gz and Parquet roll-ups. Maintained privately and offered separately; the format is documented by observations.schema.json. Interested? Open an issue.
